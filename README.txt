MIT_Covid19_Datathon
Time 8

O objetivo desse projeto era identificar padrões demográficos relacionados a infecção por COVID-19 em NYC por meio de análise exploratória de dados e feature engineering.
Utilizamos os datasets: 
NYC COVID-19 Data (cases and testings ZIP code level), disponível em https://github.com/nychealth/coronavirus-data, para esses dados fizemos a progressão no tempo pegando dados de todos os commits do repositório
Demographic Statistics By Zip Code, disponível em https://data.cityofnewyork.us/City-Government/Demographic-Statistics-By-Zip-Code/kku6-nxdu

Foram feitas análises de feature importance baseadas em regressão linear,
Growth model fitting
e Análise de Trajetória
